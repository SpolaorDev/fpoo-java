O que é Java?

Java é uma linguagem de programação rápida, segura e confiável para codificar tudo, desde aplicações móveis e software empresarial até aplicações de big data e tecnologias do servidor.

While

A declaração while executa uma instrução ou um conjunto de instruções ENQUANTO uma condição é verdadeira.

Matrizes

Uma matriz é uma estrutura de dados em Java que permite armazenar uma coleção de valores de um mesmo tipo em uma estrutura de "tabela" que é composta por linhas e colunas. Cada célula da matriz é identificada por sua posição em relação à linha e à coluna que ela ocupa.

veja um exemplo
Estrutura de Decisão

A estrutura de decisão “if” ou instrução de seleção única constitui uma construção simples que permite que uma instrução ou um bloco de instruções seja executado após a análise de determinada condição que deverá ser satisfeita para que a execução ocorra.
